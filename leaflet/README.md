# Leaflet Map Integration

Lightweight, mobile-friendly open-source mapping library integrated with GoodData.

## Key Dependencies

```json
{
  "@gooddata/sdk-ui": "^11.x",
  "@gooddata/sdk-backend-tiger": "^11.x",
  "@gooddata/sdk-model": "^11.x",
  "leaflet": "^1.9.x",
  "react-leaflet": "^4.2.x",
  "react": "^18.3.x",
  "typescript": "^5.x"
}
```

## Features

- Free OpenStreetMap tiles (no API key needed)
- Interactive markers and popups
- Dynamic coloring based on data values
- React-Leaflet bindings

## Run Locally

```bash
cd leaflet
npm install
npm start
```
